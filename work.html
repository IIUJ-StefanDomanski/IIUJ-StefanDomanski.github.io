My work

Some projects of mine I've done outside the university:

Snake4D - the first and only SNAKE IN 4TH DIMENSION. You're moving in 3 dimension + can change color (q,w,e).

Snake - simple snake game written with pure JS with the use of Queue(). My first ever steps in web technologies.

Admin's Revange - classic platformer, You have to collect dialnetes, 23 different levels to pass. Made during the IIIrd edition of Dialnet Masters competition.

Catch the Face - this one is very old and also very exhausting. The great pain of index finger guaranteed!

Ping Pong v1.1 - also old one. The aim was to code a game as fast as it is possible. The result was a clone of PONG coded in less than 30 minutes with both bot and competitive play. Updated version consist in online play through the TCP/IP protocol. It took 60 minutes to code the online multiplayer gametype.

Super Mario 3D Beta - a test to bring old Mario into 3D.

Maszyna BF - Real developer's tool to code in BF programming language! J2ME 2.0 requied (which is the phone manufactured in 2005+)

MiniBF - this is the simplest BF interpreted, it works on literally any JAVA device and weighs 2 kB.

Instrumenty J2ME - that one was a legendary app in my high school. This is a set of applications, written in J2ME, which allows you to simulate piano and percussion. It works on hardly any mobile, even in 2008 :)

Julki - A simple variation of a known logic game for J2ME phones.

Another cartoon for children - A short 3D animation showing basic character rigging.
